What I have Worked On

This week, we learned how to use WebSockets in Flask to create real-time, bidirectional communication between the client and server. This allows both the client and server to send data to each other continuously without having to refresh the page. We used the Flask-SocketIO library to set up a WebSocket-enabled server and connected it to a SocketIO client running in the browser. We created a live chat feature where each user's message is shown instanly to other users. In addition, we combined WebSockets with p5.js to create a collaboraive visual canvas where multiple users can add flowers in real time. It was really interesting to see how real time data can be synchronized across users which creates more interactive experience.
